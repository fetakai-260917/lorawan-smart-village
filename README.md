# LoRaWAN Smart Village System (by SEEDs LLP/D-lab) 

## Overview
This project demonstrates the design and deployment of a LoRaWAN-based IoT system for smart village applications.

## Architecture
Edge Devices → LoRaWAN Gateway → ChirpStack → MQTT → Node-RED → InfluxDB → Grafana/Node-RED Dashbord

## Features
- Real-time environmental monitoring
- GPS tracking
- Alert system
- Edge-based computer vision (wildlife detection)

## Technologies
- ChirpStack
- MQTT
- Node-RED
- InfluxDB
- Grafana
- Docker
- Raspberry Pi

## Notes
This repository is based on a real deployment. Sensitive data and credentials have been removed.
